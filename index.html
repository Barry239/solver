Solución paso a paso

Teniendo la ecuación homogénea

ay'' + by' + cy = 0

proponemos que

y = erx

por tanto, al derivar, tenemos que

y' = rerx, y'' = r2erx

Sustituyendo en la ecuación homogénea y resolviendo

ar2erx + brerx + cerx = 0

(ar2 + br + c)erx = 0

ar2 + br + c = 0

ar2 + br + c = 0

r1 = j, r2 = k

Al sustituir en la propuesta

y1 = ejx, y2 = ekx

Por lo tanto, tenemos que la solución general es

y = C1ejx + C2ekx
